SharePoint List Schema (for v3)
Create a list called SNF_Weekly_Tracker_v3
Required columns:
QuarterKey → Single line of text (e.g. "Q1 2026") WeekEnd → Date only FacilityName → Single line of text State → Choice (IL, IN) CMI → Number (2 decimals) MedARate → Number (0 decimals) GG15_16 → Number (0 decimals) hPPD → Number (2 decimals) Notes → Multiple lines of text Title → auto: FacilityName + :: + WeekEnd (dedup key)
Each weekly cell = one list item. Title field is used as a dedup key so re-syncing updates existing rows instead of duplicating.
